# mlp-california-housing-price-prediction
Multilayer Perceptron (MLP) model for predicting house prices using the California Housing dataset. A deep learning regression project for learning and research.

### Step 1: Clone the project

```bash
git clone https://github.com/zombieTDV/mlp-california-housing-price-prediction.git
```

### Step 2: Create Virtual Environment

```bash
python -m venv .venv
```

Activate the virtual environment:

- On Windows (PowerShell):

```bash
.venv\Scripts\Activate.ps1
```

- On macOS/Linux:

```bash
source .venv/bin/activate
```

---

#### Download library requirements

```bash
pip install -r requirements.txt
```
